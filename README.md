# Personal Terminal Homepage

A hacker-terminal-style personal homepage with Matrix rain background, typing animation, and interactive command line.

## Features

- 🖥️ Terminal aesthetic with green-on-black theme
- 🌧️ Matrix rain canvas background
- ⌨️ Interactive command line (type `help` to start)
- 📜 Typing animation on load
- 📊 ASCII art banner + skill bars
- 📱 Fully responsive for mobile devices
- ⚡ Zero dependencies, pure HTML/CSS/JS

## Commands

| Command | Description |
|---|---|
| `help` | Show available commands |
| `whoami` | About me |
| `skills` | Tech stack |
| `projects` | Project list |
| `contact` | Contact info |
| `clear` | Clear terminal |
| `banner` | Show ASCII art again |
| `date` | Current date/time |
| `echo` | Echo test |

## Usage

Open `index.html` in your browser, or deploy to GitHub Pages.

## Customization

Edit the HTML to replace placeholder content with your own:
- Name & tagline in the banner section
- Skills in the `cat skills.txt` block
- Projects in the `ls projects/` block
- Contact info in the `cat contact.json` block
